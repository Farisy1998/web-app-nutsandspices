Read me to run the project
---------------------------------------

Requirements to install
-------------------------
1. Nodejs
2. STS IDE
3. VS code IDE
4. Java 17
5. MySQL 8.0 and MySQL workbench

How to run
---------------------
1. Download the project.
2. Open VS code inside frontend folder.
3. Install node_module using "npm install" command.
4. Import backend folder as maven project inside STS IDE.
5. Create a local connection in MySQL workbench using "root" as username and password.
6. Create a new database in side MySQL as "web_db".
7. Create tables based on the entity classes inside the "src/main/java" entities package.
8. After creating database and tables, run the backend project as Spring boot app from STS.
9. From VS code create a new terminal and run the command "npm start".
